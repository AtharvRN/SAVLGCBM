# Unified Config Examples

These JSON files are example run configs for the unified codebase.

## Current status

- `vlg_cbm` configs are intended to be runnable now.
- `lf_cbm` configs are intended to be runnable now.
- `salf_cbm` and `savlg_cbm` configs are planning examples for the next porting stages.

## Convention

File name pattern:

- `<dataset>_<model_name>.json`

Datasets:

- `cifar10`
- `cifar100`
- `cub`
- `places365`
- `imagenet`

Models:

- `vlg_cbm`
- `lf_cbm`
- `salf_cbm`
- `savlg_cbm`
